Welcome to BINARY: Brain Image graNARY

Oct. 19, 2020

This is an Insight Data Engineering minimal viable project that provides a RESTful API for retrieving brain MRI images. So far, it contains a couple of datasets extracted from the following sources:

  • TCIA: ACRIN-DSC-MR-Brain (ACRIN ACRIN 6677/RTOG 0625)
  • Human Connectome Project (1200 Subjects Data Release)

List of predefined queries:

Click on the button to open the dropdown list of queries. Please note that it takes about 90 seconds to compress the MRI images from about 1000 subjects (total about 3 GB) and the download will then start automatically.
